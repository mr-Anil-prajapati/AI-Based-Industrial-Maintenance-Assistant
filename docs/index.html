README Summary

The software is designed as a read-only industrial maintenance assistant. It must never modify PLC programs, send commands to PLCs, change machine parameters, or control machines.

It uses local Retrieval Augmented Generation with imported documents such as .pdf, .docx, .txt, and .md files. The desktop version is built for offline use on Windows and is packaged with a local model runtime.

What It Does

  • Reads machine manuals, alarm tables, and maintenance guides
  • Analyzes PLC signal snapshots in read-only mode
  • Explains likely faults and recommended checks
  • Supports motors, pumps, compressors, conveyors, and PLC troubleshooting

Desktop Features

  • Floating chat window
  • Collapsible assistant panel
  • Document import
  • Clipboard-based troubleshooting context
